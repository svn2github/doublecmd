Клавиатурные сочетания Double Commander

Содержание
1. Главное окно
2. Встроенный просмоторщик
3. Встроенный редактор

1. Главное окно

Клавиша Действие
F1 открыть окно "О программе..."
F2 переименовать файл/каталог под курсором
F3 1. открыть выделенные файлы в просмоторщике
2. открыть каталог под курсором
F4 открыть файл под курсором в редакторе
F5 копировать файлы/каталоги
F6 переименовать или переместить файлы
F7 создать новый каталог
F8, Del удалить файлы/каталоги
F9 запустить терминал
Alt+F1 открыть список дисков для левой панели
Alt+F2 открыть список дисков для правой панели
Alt+F4, Alt+X выход
Alt+F5 упаковать выбранные файлы/каталоги
Alt+F7 поиск файлов/каталогов
Alt+F9 распаковать все файлы из выделенных архивов
Alt+Enter показать диалог свойств файла/каталога
Alt+Shift+Enter Показать размеры всех папок
Alt+Delete Уничтожить файл/каталог
Shift+F4 создать новый текстовый файл и загрузить его в редактор
Shift+F5 копировать файлы (с переименованием) в тот же каталог
Shift+F6 переименовать файл/каталог под курсором
Shift+Enter выполнить приложение в терминале (терминал берется из настроек)
Ctrl+F3 сортировать по имени
Ctrl+F4 сортировать по типу
Ctrl+F5 сортировать по дате/времени
Ctrl+F6 сортировать по размеру
Ctrl+H открыть историю уже посещённых каталогов
Ctrl+D открыть меню избранных каталогов
Ctrl+L подсчитать занимаемое место (для выбранных файлов/каталогов)
Ctrl+M инструмент группового переименования
Ctrl+R обновить содержимое текущей панели
Ctrl+ ↑ открыть каталог под курсором в новой вкладке
Ctrl+ ↓ открыть историю команд командной строки
Ctrl+Enter копировать имя файла в командную строку
Ctrl+P копировать путь в командную строку
Ctrl+Shift+Enter копировать полное имя файла в командную строку
Ctrl+U поменять местами панели (активные вкладки)
Ctrl+С копировать в буфер обмена
Ctrl+X вырезать в буфер обмена
Ctrl+V вставить из буфера обмена
Ctrl+T создать новую вкладку для текущей панели
Ctrl+W закрыть активную вкладку в текущей панели
Ctrl+Z создать/изменить комментарий к файлу
Ctrl+PgDown 1. аналогично клавише Enter
2. открыть архив (включая самораспаковывающиеся архивы)
Ctrl+PgUp перейти в родительский каталог
Ctrl+Num + выделить все
Ctrl+Num - снять все выделение
Num + выделить группу по шаблону
Num - снять выделение группы по шаблону
Num * инвертировать выделение
Shift+Num + выделить все файлы в текущем каталоге с расширением как у файла под курсором
Shift+Num - снять выделение со всех файлов в текущем каталоге с расширением как у файла под курсором
Shift+F2 установить фокус в командную строку
Tab переключиться на другую файловую панель
← перейти в родительский каталог (если активирован режим "Навигация в стиле Lynx")
→ Перейти в выбранный каталог (если активирован режим "Навигация в стиле Lynx")
Enter 1. если командная строка непуста, выполнить её
2. если курсор на каталоге/архиве: открыть этот каталог/архив
3. если курсор на файле программы: запустить программу
4. если курсор на обычном файле: запустить ассоциированную программу
5. если курсор на файле внутри архива: показать диалог свойств упакованного файла
Insert выделить файл/каталог
Backspace перейти в родительский каталог
Пробел 1. инвертировать выделение файла
2. инвертировать выделение каталога, с подсчетом занимаемого места
Буква 1. переместить курсор в командную строку и ввести букву там.
2. в режиме быстрого поиска "Только буква": быстрый поиск имени файла/каталога (начинающегося с указанных букв) в текущем каталоге.
Правая клавиша мыши показать контекстное меню с настраиваемым списком действий (из файла doublecmd.ext)

наверх

2. Встроенный просмоторщик

Клавиша Действие
N следующий файл, если было открыто несколько файлов
P предыдущий файл, если было открыто несколько файлов
Esc, Q (с любой комбинацией Ctrl, Shift, Alt) выход
1 только текст
2 двоичный (фиксированная длина строки)
3 шестнадцатеричный
4 только текст, с разрывами строк
6 графика
7 использовать плагины

наверх

3. Встроенный редактор

Клавиша Действие
F2, Ctrl+S сохранить
Esc выход

наверх
