Настройки файла doublecmd.ini

Большинство настроек файла doublecmd.ini может быть изменено из диалоговых окон настройки. Прямое редактирование файла может потребоваться в редких случаях.

[Configuration] Основные настройки
CaseSensitiveSort=0 0: Case sensitive sort OFF
1: Case sensitive sort ON
CopyBlockSize=64 Устанавливает размер блока при копировании (в Кб=1024 байт). По умолчанию 64 (=65536 байт).
CustomDriveIcons=0 Использовать свои иконки для дисков (из %commander_path%\pixmaps\XxX\devices\) вместо системных (только Windows)
0: Использовать системные иконки
1: Использовать пользовательские иконки
DirTabLimit=32 Максимальное число символов на заголовке вкладки
DriveBlackList Список дисков которые не будут отображаться на панели дисков и в меню дисков (; (точка с запятой) — разделитель)
UseExtEdit=0 0: Использовать встроенный редактор
1: Использовать внешний редактор (см. ExtEdit)
UseExtView=0 0: Использовать встроенный просмоторщик
1: Использовать внешний просмоторщик (см. ExtView)
UseExtDiff=0 0: Использовать внутренний диффер
1: Использовать внешний диффер (см. ExtDiff)
ExtEdit путь к внешнему редактору для F4
ExtView путь к внешнему просмоторщику для F3
ExtDiff путь к внешнему дифферу
Font.Name=default Шрифт для файловых панелей
Font.Size=10 Размер шрифта для файловых панелей
Font.Style Стиль шрифта для файловых панелей
Maximized=0 Состояние окна
0: обычное окно
1: окно, развёрнутое на весь экран
MouseSelectionEnabled=1 Разрешить выделение мышью
0: нет
1: да
QuickSearchMatchBeginning=1 0: A search string like "double" will be read as "*double"
1: Exact match: typing "cmd" will find only "cmd.ini", but not "doublecmd.ini"
QuickSearchMatchEnding=1 0: Typing a dot is read as *. , "double.i" finds also "doublecmd.ini"
1: Exact match: typing "double.i" finds only "double.ini", but doesn't find "doublecmd.ini"
ScrollMode=0 Режим прокрутки мышью
0: Построчно с движением курсора
1: Построчно
2: Постранично
ShowIcons=1 Показывать иконки в файловых панелях
0: Нет
1: Да
ShowSystemFiles=0 Показывать скрытые/системные файлы
0: Нет
1: Да
UseIniInProgramDir=0 Месторасположение конфигурационных файлов
0: Домашний каталог пользователя
1: Каталог программы
[Colors] Настройки цветов
[PackerPlugins] Список установленных архиваторных плагинов
bz2=0,%commander_path%\plugins\unbz2.wcx Плагин для работы с bz2-архивами. 0 — сумма возможностей плагина
[Content Plugins] Список и настройки контентных плагинов
PluginCount=2 Количество установленных контентных плагинов
Plugin1Name=rpm_wdx Название первого контентного плагина
Plugin1Detect=EXT="RPM" Первый плагин срабатывает для расширения RPM
Plugin1Path Путь к первому контентному плагину
[ColumnsSet] Список доступных наборов колонок
ColumnsSetCount=2 Количество достуных наборов колонок
ColumnsSet1Name=Default Первый наборок колонок — Default
[lefttabs] Список вкладок левой панели
[righttabs] Список вкладок правой панели
