Intenal commands

Command Description Keys
by default
cm_About Show program About information F1
cm_AddPathToCmdLine Put path to command line Ctrl+P
cm_CalculateSpace Calculate occupied space Ctrl+L
cm_ChangeDirToRoot Go to root directory ("/" in GNU/Linux)
cm_CheckSumCalc Calculate file checksum (MD5 or SHA1)
cm_CheckSumVerify Verify MD5 or SHA1 checksum
cm_CompareContents Open compare by contents dialog
cm_ContextMenu Shows context menu for files and folders
cm_Copy Copy items from source to target F5
cm_CopyFullNamesToClip Copy items full names to clipboard Ctrl+Alt+C
cm_CopyNamesToClip Copy items names to clipboard Ctrl+Alt+X
cm_CopySamePanel Copy item to the same panel Shift+F5
cm_CopyToClipboard Copy selected text, file or directory to clipboard Ctrl+C
cm_CutToClipboard Cut selected text, file or directory to clipboard Ctrl+X
cm_Delete Delete selected item F8
cm_DirHistory Invoke dir history pop up menu Ctrl+H
cm_DirHotList Directory hotlist Ctrl+D
cm_DriveContextMenu Call context menu of some drive. Mount point sets at parameters, e.g. (/media/cdrom)
cm_Edit Open file in internal editor F4
cm_EditComment Create or edit comment for file/directory Ctrl+Z
cm_EditNew Create new file, and open it at the editor (see configuration) Shift+F4
cm_Exchange Swap file panels Ctrl+U
cm_Exit Close Double Commander Alt+X
cm_ExtractFiles Unpack one or some selected archives Alt+F9
cm_FileAssoc Open file associations configuration
cm_FileLinker Combine splitted files with extension .split
cm_FileProperties Call item properties (size, data, atributes, etc.) Alt+Enter
cm_FileSpliter Split files dialog
cm_FocusCmdLine Set focus to the command line Shift+F2
cm_HardLink Create hard link
cm_HelpIndex Open Double Commander help index page
cm_Keyboard Open keyboard shortcuts help
cm_LeftEqualRight Goto same directory at inactive panel Alt+Z
cm_LeftOpenDrives Open drives list for left panel Alt+F1
cm_MakeDir Create new directory F7
cm_MarkCurrentExtension Mark files with same extension, as file under cursor
cm_MarkInvert Invert selection
cm_MarkMarkAll Mark all at the panel Ctrl+A
cm_MarkMinus Unselect marked items by mask
cm_MarkPlus Select items by mask
cm_MarkUnmarkAll Unselect all marked items F1
cm_MultiRename Open multi rename tool dialog Ctrl+M
cm_NewTab Create new tab for opened direcory Ctrl+T
cm_NextTab Change to next tab Ctrl+Tab
cm_Open Open file or program under cursor Enter
cm_OpenArchive Try to open file as archive Ctrl+PgDown
cm_OpenDirInNewTab Open directory at new tab, but dont switch to it
cm_OpenVFSList Open VFS list (depends on installed plugins)
cm_Options Open configuration dialog
cm_PackFiles Pack items to archive Alt+F5
cm_PanelSpliterPerPos Set panels width. E.g. 50% —equal size
cm_PasteFromClipboard Paste data from clipboard Ctrl+V
cm_PrevTab Change to previous tab Shift+Ctrl+Tab
cm_QuickSearch Quick search at directory Ctrl+S
cm_Refresh Refresh panels Ctrl+R
cm_RemoveAllTabs Close all tabs
cm_RemoveTab Close active tab Ctrl+W
cm_Rename Rename or move items F6
cm_RenameOnly Rename files in the same directory Shift+F6, F2
cm_ReverseOrder Invert sorting items at the directory
cm_RightOpenDrives Open drives list for right panel Alt+F2
cm_RunTerm Run terminal F9
cm_Search Open Search dialog Alt+F7
cm_ShowButtonMenu Create context menu. Need parameter
cm_ShowCmdLineHistory Show history of all commands at command line Ctrl+7
cm_ShowMenu Open Double Commander main menu
cm_ShowSysFiles Show hidden and system files
cm_SortByAttr Sort items at file panels by attributes
cm_SortByDate Sort items at file panels by date Ctrl+F5
cm_SortByExt Sort items at file panels by extension Ctrl+F4
cm_SortByName Sort items at file panels by name Ctrl+F3
cm_SortBySize Sort items at file panels by size Ctrl+F6
cm_SymLink Create symlink to file/directory
cm_TargetEqualSource Goto same directory at inactive panel Alt+Z
cm_ToggleLockDcaTab Lock current tab, but able to change directory
cm_ToggleLockTab Lock tab
cm_TransferLeft Open directory under cursor at the left panel Ctrl+Left
cm_TransferRight Open directory under cursor at the right panel Ctrl+Right
cm_UnmarkCurrentExtension Unmark items by mask
cm_View Open file at viewer program F3
cm_VisitHomePage Go to Double Commander home page
cm_Wipe Wipe file Alt+Del
