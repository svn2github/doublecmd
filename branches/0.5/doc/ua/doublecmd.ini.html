Налаштування файлу doublecmd.ini

Більшість налаштувань файлу doublecmd.ini може бути змінено з діалогових вікон налаштувань. Пряме редагування файлу може знадобитися дуже рідко.

[Configuration] Основні налаштування
CaseSensitiveSort=0 0: Case sensitive sort OFF
1: Case sensitive sort ON
CopyBlockSize=64 Встановлює размір блоку при копіюванні (в Кб=1024 байт). За замовчуванням 64 (=65536 байт).
CustomDriveIcons=0 Використовувати своі іконки для дисків (з %commander_path%\pixmaps\XxX\devices\) замість системних (тільки у Windows)
0: Використовувати системні іконки
1: Використовувати іконки користувача
DirTabLimit=32 Максимальне число символів у заголовку вкладки
DriveBlackList Список дисків які не будуть відображатися на панелі дисків і в меню дисків (; (крапка з комою) — разділювач)
UseExtEdit=0 0: Використовувати вбудований редактор
1: Використовувати зовнішній редактор (див. ExtEdit)
UseExtView=0 0: Використовувати вбудований переглядач
1: Використовувати зовнішній переглядач (см. ExtView)
UseExtDiff=0 0: Використовувати внутрішній порівнювач
1: Використовувати зовнішній порівнювач (див. ExtDiff)
ExtEdit шлях до зовнішнього редактору для F4
ExtView шлях до зовнішнього переглядача для F3
ExtDiff шлях до зовнішнього порівнювача
Font.Name=default Шрифт для файлових панелей
Font.Size=10 Размір шрифту для файлових панелей
Font.Style Стиль шрифту для файлових панелей
Maximized=0 Стан вікна
0: звичайне вікно
1: вікно, розгорнуте на весь екран
MouseSelectionEnabled=1 Дозволити виділення мишкою
0: нет
1: да
QuickSearchMatchBeginning=1 0: A search string like "double" will be read as "*double"
1: Exact match: typing "cmd" will find only "cmd.ini", but not "doublecmd.ini"
QuickSearchMatchEnding=1 0: Typing a dot is read as *. , "double.i" finds also "doublecmd.ini"
1: Exact match: typing "double.i" finds only "double.ini", but doesn't find "doublecmd.ini"
ScrollMode=0 Режим прокрутки скролом
0: Порядково з рухом курсора
1: Порядково
2: Посторінково
ShowIcons=1 Показуватиь іконки у файлових панелях
0: Ні
1: Так
ShowSystemFiles=0 Показувати приховані/системні файли
0: Ні
1: Так
UseIniInProgramDir=0 Місцезнаходження конфігураційних файлів
0: Домашній каталог користувача
1: Каталог програми
[Colors] Налаштування кольорів
[PackerPlugins] Список встановлених архіваторних плагінів
bz2=0,%commander_path%\plugins\unbz2.wcx Плагін для роботи з bz2-архівами. 0 — сума можливостей плагіну
[Content Plugins] Список і налаштування контентних плагінів
PluginCount=2 Кількість встановлених контентних плагінів
Plugin1Name=rpm_wdx Назва першого контентного плагіну
Plugin1Detect=EXT="RPM" Перший плагін спрацьовує для розширення RPM
Plugin1Path Шлях до першого контентного плагіну
[ColumnsSet] Список доступних наборів колонок
ColumnsSetCount=2 Кількість достуних наборів колонок
ColumnsSet1Name=Default Перший набір колонок — Default
[lefttabs] Список вкладок лівої панелі
[righttabs] Список вкладок правої панелі
