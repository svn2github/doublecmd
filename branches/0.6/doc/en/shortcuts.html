Ctrl+PgUp Like backspace
Ctrl+Num + Select all

Num + Expand selection
Num - Shrink selection

Insert Select file or directory
Backspace Goto to the upper (parent) directory
Space 1. On file - select / deselect item
2. On directory - select / deselect item and compute space occupied in dir
alphanumeric keys 1. Write to command line.
2. If option "Letter only" enabled - quick search in the panel.
Right mouse button Show popup menu with configurable commands (as defined in file doublecmd.ext)
